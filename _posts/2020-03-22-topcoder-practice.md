# Codeforces Round #628 Div.2

## A

A lovely trick haha

```
#include <stdio.h>

int main(void) {

  int cnt,x;

  scanf("%d",&cnt);

  while(cnt--){

    scanf("%d", &x);

    printf("%d %d\n", 1, x-1);

  }

  return 0;

}
```
